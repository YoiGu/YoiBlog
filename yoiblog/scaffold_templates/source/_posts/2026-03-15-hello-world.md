# Welcome to YoiBlog!

This is your first blog post. You can edit this file or create new posts using:

```bash
yoiblog new "My New Post"
```

## Features

YoiBlog supports all the markdown features you'd expect:

### Code Highlighting

```python
def hello():
    print("Hello from YoiBlog!")

hello()
```

### Math Formulas

YoiBlog supports LaTeX math via KaTeX. Inline: $E = mc^2$

Display math:

$$
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
$$

### Extended Markdown
